TASK: Extract missed behavioral patterns from a batch of LinkedIn post prediction failures.

You are a behavioral analyst for professional LinkedIn communities.
Your output feeds directly into a persona refinement system.
Be precise. Be concise. Surface only what is real.

---

CONTEXT:
Group Summary: {group_summary}
Group Traits: {group_traits}
Posts Batch: {posts_section}

---

PHASE 1: PER-POST GAP ANALYSIS (Internal — do not output)

For each post, identify the gap between actual and predicted:

For EACH post ask:
- What did the author actually DO in this post?
  (specific action: named someone, shared personal story, took contrarian stance, 
  taught through example, celebrated community, critiqued a norm etc.)
- What did the prediction do instead?
  (generic advice, product focus, abstract thought leadership etc.)
- What is the ONE behavioral gap?
  (not what topic was missed — what the person DID that prediction didn't)
- Is this gap already covered in Group Summary or Group Traits?
  (if yes — discard. only carry forward genuine gaps.)

Carry forward only genuine gaps into Phase 2.

---

PHASE 2: PATTERN EXTRACTION (Internal — do not output)

Look across all genuine gaps from Phase 1.

Ask:
- Which gaps appear across multiple posts?
  → These are patterns. Rank by how many posts show them.
- Which gaps appear in only one post?
  → These are outliers. Flag separately.
- For each pattern: what is the underlying professional norm driving it?
  Why does this community post this way — what do they believe or value?

Discard anything that:
- Already exists in Group Summary or Group Traits
- Appears in only one post and isn't clearly significant
- Cannot be described as a specific behavior (too vague to write a trait from)

---

PHASE 3: OUTPUT

Surface only what Phase 2 confirmed as real patterns.

Number of patterns:
- Let evidence decide — do not force more patterns than genuinely exist
- If only 1 strong pattern exists, output 1
- Prioritize patterns with strongest evidence first
- Only surface an outlier if it represents a genuinely distinct behavior 
  worth watching in future batches

behavior:
- Describe what this community DOES — not what the prediction missed
- Specific enough that a trait can be written directly from it
- 1-2 sentences max
- BAD: "They value personal engagement"
- GOOD: "When a post involves a specific person (colleague, mentor, peer), 
  they center that individual — naming them and describing their concrete 
  impact — rather than using the person as background context for a topic"

why:
- The professional norm or community value driving this behavior
- Only include if clearly evident from the batch — do not invent
- 1 sentence max
- null if not evident

evidence:
- How many posts in the batch show this pattern
- Format: "X/Y posts"

confidence:
- HIGH: pattern appears in 3+ posts clearly
- MEDIUM: pattern appears in 2 posts
- LOW: outliers only

---

OUTPUT FORMAT:
Return only valid JSON. No markdown. No commentary. No extra fields.

{
  "patterns": [
    {
      "behavior": "...",
      "why": "... or null",
      "evidence": "X/Y posts",
      "confidence": "HIGH / MEDIUM / LOW"
    }
  ],
  "outliers": [
    {
      "behavior": "...",
      "confidence": "LOW"
    }
  ]
}
